## [1.0.0] - 2024-11-10

### 添加
- 实现同步指定目录下所有 Git 仓库的功能
  - 递归遍历目录，查找所有 Git 仓库
  - 检查每个 Git 仓库是否有远程仓库
  - 支持拉取所有分支或指定分支
- 创建 `README.md` 文件，详细说明项目使用方法
- 创建 `LICENSE` 文件，采用 MIT 许可证
- 创建 `CHANGELOG.md` 文件，记录项目变更

### 修改

### 修复
